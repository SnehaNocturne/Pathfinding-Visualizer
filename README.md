Pathfinding Visualizer in Java using BFS to find the shortest path in a grid with obstacles.
# Pathfinding Visualizer 🚀

This project implements a pathfinding algorithm using BFS in Java.

## Features
- Finds shortest path
- Shows steps count
- Displays path on grid

## Algorithm Used
- Breadth First Search (BFS)

## Output
Displays path using '*' symbol

## Future Improvements
- GUI Visualization
- Dijkstra Algorithm
- A* Algorithm
